# Logpsy 🤖🩺

Logpsy is an SRE incident analysis assistant trained to perform cautious, structured root-cause analysis on logs and metrics.

## Repository Structure

```text
logpsy/
  README.md                 # Main overview documentation
  LICENSE                   # MIT License
  requirements.txt          # Python dependencies

  data/
    logpsy_gold_dataset_sample.jsonl       # Sample gold-standard incidents
    logpsy_eval_cases.jsonl                # Verification test cases
    dataset_audit_report.md                # Validation report on dataset constraints

  examples/
    sample_inputs.md                       # Log sequences for testing
    sample_outputs.md                      # JSON output returned by logpsy-7b-v2

  model/
    README.md                              # Model specs and Ollama import instructions
```

## Running the Model

Please refer to [`model/README.md`](model/README.md) for instructions on how to load the GGUF model and run local inference using Ollama.

## Examples

Check the [`examples/`](examples/) directory for sample input log sequences and the corresponding cautious JSON analysis outputs generated by the fine-tuned model.
